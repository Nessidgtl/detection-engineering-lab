# Detection Approach

### For each scenario:

1. **Understand the behavior chain**  
   Break the attack into clear, sequential actions

2. **Simulate or reason through execution**  
   Reproduce the chain of events or walk through it logically

3. **Identify observable artifacts**  
   Processes, logs, authentication events, network activity

4. **Build detection logic**  
   Translate behavior into queries (EQL/KQL)

5. **Stress-test assumptions**  
   Consider bypasses, noise, and edge cases

6. **Refine the detection**  
   Adjust based on gaps, false positives, and real life issues
